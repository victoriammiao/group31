Express
Express.js 是一个基于 Node.js 的 Web 应用框架，用于构建 Web 服务器和 API。它是一个非常轻量级且灵活的框架，被广泛用于创建 RESTful API 和 Web 应用。Express 提供了丰富的功能来简化路由管理、请求和响应处理等常见任务。

特点
轻量和灵活：Express 是一个极简的框架，开发者可以自由选择中间件和工具。
中间件支持：Express 的核心是中间件架构，用户可以通过添加中间件处理 HTTP 请求，进行验证、授权、日志记录、数据解析等。
路由系统：提供了强大的路由系统，可以定义不同路径和请求方法（如 GET、POST）下的处理逻辑。
适合 RESTful API：适用于创建 RESTful API，可以轻松与前端（如 Vue、React）集成。
